

NFT Gallery

> display your NFT collection <


this gallery is made for collections made with hashlips script
see this video:
https://www.youtube.com/watch?v=vFY_E3IP6OU

How to use:

create list of your images like this: 

[
  "https://i.postimg.cc/vmVFdV65/1.png",
  "*yourimageURL*",
  "*yourimageURL*",
  etc...
]


paste your metadata from your collection in the file ./src/meta.tsx

and voila,

Congradulations!

you have an NFT image Gallery!


connect with a Netlify account to host it
